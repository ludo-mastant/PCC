PORTFOLIO PCC - BTS SIO SLAM

Structure du projet :
- index.html : structure du portfolio
- style.css : mise en page, responsive, thème clair/sombre
- script.js : menu mobile, filtre des projets, bouton retour haut, copie email, animations
- assets/certifications : images de certifications à remplacer
- docs : documents téléchargeables à remplacer par tes vraies fiches

À modifier rapidement avant l’oral :
1. Dans index.html, remplace "Prénom NOM" par ton nom.
2. Remplace "prenom.nom@email.fr" dans la section contact.
3. Remplace les textes de projet par tes vrais éléments.
4. Mets tes vrais certificats dans assets/certifications.
5. Mets tes vrais documents dans docs.
6. Si tu mets un PDF, remplace le href du bouton par exemple :
   href="docs/ma-certification.pdf"

Conseil pour l’oral :
Utilise le portfolio comme support. Ne lis pas tout. Explique :
- le besoin métier
- les choix techniques
- les compétences BTS associées
- les preuves disponibles
- les limites et améliorations possibles
